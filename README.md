# FilmFusion
Front-end приложение для поиска и просмотра фильмов

# Cтек технологий
- React + React Hooks
- TypeScript
- Redux Toolkit
- Sass, Tailwind CSS

# Запуск приложения 
1. `yarn install` - установите зависимости
2. `yarn run pretest` - установите types для работы с API
3. `yarn run dev` - запустите проект
